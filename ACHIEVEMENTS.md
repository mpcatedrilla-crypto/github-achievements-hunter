# 🏆 GitHub Achievements Complete Guide

## Achievement Details

### 🦈 Pull Shark
**Tiers:** 🥉 Bronze (2) | 🥈 Silver (16) | 🥇 Gold (128)

**How to Unlock:**
- Merge pull requests on any repository
- Can be your own repository
- Fastest way: Create a repo, make branches, open and merge your own PRs

**Automation:**
```bash
./scripts/pull-shark.sh 128
```

---

### 🤪 YOLO
**Tiers:** 🥇 Gold only (1)

**How to Unlock:**
- Merge a pull request without review
- Requires admin access to repository
- Use `gh pr merge --admin` flag

**Automation:**
```bash
./scripts/yolo.sh
```

---

### 🧠 Galaxy Brain
**Tiers:** 🥉 Bronze (1) | 🥈 Silver (16) | 🥇 Gold (32)

**How to Unlock:**
- Answer discussions in GitHub Discussions
- Must be marked as "Answer" by question author
- Enable Discussions in repo settings first

**Steps:**
1. Go to Settings → Features → Enable Discussions
2. Create categories (Q&A recommended)
3. Ask questions and answer them yourself
4. Mark your answer as the answer

---

### ⭐ Starstruck
**Tiers:** 🥉 Bronze (16) | 🥈 Silver (128) | 🥇 Gold (512)

**How to Unlock:**
- Create a repository that gets stars
- Stars must come from unique users
- Only counts for repos you created

**Tips to Get Stars:**
1. **Developer Tools**
   - CLI utilities
   - VS Code extensions
   - GitHub Actions

2. **Templates & Starters**
   - Project boilerplates
   - Config files
   - Starter kits

3. **Learning Resources**
   - Cheatsheets
   - Tutorials
   - Awesome-lists

4. **Self-Promotion**
   - Reddit: r/webdev, r/github, r/programming
   - Twitter/X with relevant hashtags
   - Dev.to articles
   - Submit to awesome-lists

**Tracking:**
```bash
./scripts/starstruck.sh
```

---

### ⚡ Quickdraw
**Tiers:** 🥇 Gold only (1)

**How to Unlock:**
- Close an issue or pull request within 5 minutes of opening
- Works on any repository you have access to

**Automation:**
```bash
./scripts/quickdraw.sh
```

---

### 👥 Pair Extraordinaire
**Tiers:** 🥉 Bronze (1) | 🥈 Silver (16) | 🥇 Gold (32)

**How to Unlock:**
- Co-author commits on merged pull requests
- Commit must include `Co-authored-by:` trailer
- PR must be merged

**Commit Format:**
```
Commit message

Co-authored-by: Name <email@example.com>
Co-authored-by: Name2 <email2@example.com>
```

**Automation:**
```bash
./scripts/pair-extraordinaire.sh 16
```

---

### 💖 Public Sponsor
**Tiers:** 🥇 Gold only (1)

**How to Unlock:**
- Sponsor any GitHub user through GitHub Sponsors
- Minimum $1/month sponsorship
- Must be a public sponsorship

**Steps:**
1. Go to https://github.com/sponsors
2. Find a developer to sponsor
3. Choose tier (minimum $1)
4. Make it public (not private)

## 🎓 Best Practices

1. **Don't Spam**
   - GitHub may flag excessive automated activity
   - Space out your achievements
   - Create meaningful commits

2. **Use Private Repos First**
   - Test scripts on private repos
   - Avoid noise in public contribution graphs

3. **Combine Achievements**
   - One PR can count for multiple achievements
   - Example: Co-authored PR merged without review = Pull Shark + Pair Extraordinaire + YOLO

4. **Track Progress**
   - View your achievements: https://github.com/YOUR_USERNAME?tab=achievements
   - Check profile README to display them
